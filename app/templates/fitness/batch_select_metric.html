About Batch Recording

Batch recording allows you to efficiently record the same fitness metric for multiple players at once. This is useful for recording data collected during team training sessions.

Tips:

  • You can leave fields blank for players who didn't participate
  • All records will be timestamped with the current date and time
  • You can add optional notes for each player's record
  • Use the "Fill with previous values" option to pre-populate fields with each player's most recent data

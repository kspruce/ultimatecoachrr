Database Backup & Restore Guide

This tool allows you to create backups of your database and restore them when needed.

Creating Backups:
  1. Go to the Export Data tab
  2. Use Download All Tables (JSON Format) for backups you plan to restore later
  3. Use Download All Tables (Excel Format) for data you want to view or analyze
  4. The downloaded ZIP file contains all your database tables
Restoring Backups:
  1. Go to the Import Data tab
  2. Upload your JSON format backup ZIP file
  3. Check "Clear existing data" if you want to replace all current data
  4. Confirm and click "Import JSON Backup"
  5. Note: Only JSON format backups can be imported

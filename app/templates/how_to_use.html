Create a collection of drills to use in your sessions:

  1. Go to Session Plans > Drill Library
    2. {% if current_user.is_admin %}
  2. Select "Add Basic Drill"
  3. Define drill parameters:
    • Name and description
    • Setup instructions
    • Skills developed
    • Variations
  4. Save to your drill library
    5. {% else %}
  5. Browse existing drills
  6. Note: Only administrators can add new drills
    7. {% endif %}
Drill Library
Creating Session Plans

The Clip Library helps you organize video footage for analysis, teaching, and review.

Adding Clips
  1. Navigate to the Clips section
    2. {% if current_user.is_admin %}
  2. Select "Add New Clip"
  3. Upload video or provide a link (YouTube, Vimeo, etc.)
  4. Add metadata:
    • Title and description
    • Related game/point (if applicable)
    • Featured players
    • Tags for categorization
    5. {% else %}
  5. Browse existing clips
  6. Note: Only administrators can add new clips
    7. {% endif %}
Annotating Clips
  1. Open a clip for viewing
    2. {% if current_user.is_admin %}
  2. Use the annotation tools to:
    • Add timestamps for key moments
    • Draw on the video to highlight movements
    • Add text notes to explain concepts
  3. Save annotations for team review
    4. {% else %}
  4. View existing annotations added by coaches
  5. Note: Only administrators can add or edit annotations
    6. {% endif %}
Tip

Create collections of clips for specific purposes: teaching a concept, highlighting a player's development, or preparing for an opponent.

Scouting Reports

Opponent Analysis & Preparation

Create detailed scouting reports to prepare your team for upcoming opponents.

Creating Team Reports
  1. Go to the Scouting section
    2. {% if current_user.is_admin %}
  2. Select "New Report"
  3. Add team information:
    • Team name and background
    • Offensive and defensive systems
    • Key plays and formations
    • Team tendencies and patterns
  4. Save and share with your team
    5. {% else %}
  5. Browse existing scouting reports
  6. Note: Only administrators can create new reports
    7. {% endif %}
Player Scouting
  1. Within a team report, add key players
    2. {% if current_user.is_admin %}
  2. For each player, document:
    • Position and role
    • Throwing preferences
    • Cutting patterns
    • Defensive style
    • Matchup suggestions
  3. Link relevant video clips if available
    4. {% else %}
  4. View player information added by coaches
  5. Note: Only administrators can add or edit player scouting information
    6. {% endif %}
Tip

Focus on actionable intelligence: what specific adjustments should your team make based on this scouting information?

Common Workflows

Here are some typical workflows that combine multiple features of Ultimate Coach:

Game Day Workflow
    {% if current_user.is_admin %}
  1. Create a new game in the Games section
  2. Record points as they happen during the game
  3. Input detailed statistics after each point or post-game
  4. Review game stats and share insights with the team
  5. Add notable clips to the Clip Library for further analysis
    6. {% else %}
  6. View game details in the Games section
  7. Follow along with points as they're recorded
  8. Review game stats after they're entered by administrators
  9. View clips added to the Clip Library by administrators
  10. Review insights shared by coaches
    11. {% endif %}
Practice Planning Workflow
  1. Review recent game statistics to identify areas for improvement
    2. {% if current_user.is_admin %}
  2. Create a session plan focusing on these areas
  3. Add relevant drills from your drill library
  4. Include playbook elements to practice
  5. Track attendance and player progress
    6. {% else %}
  6. View session plans created by coaches
  7. Review drills that will be used in practice
  8. Study playbook elements that will be practiced
  9. Check your attendance record
    10. {% endif %}
Tournament Preparation Workflow
    {% if current_user.is_admin %}
  1. Create scouting reports for upcoming opponents
  2. Review and update your playbook for specific matchups
  3. Create focused session plans to prepare
  4. Add tournament schedule to the system
  5. During the tournament, track games, points, and stats
    6. {% else %}
  6. Review scouting reports created by coaches
  7. Study playbook elements relevant to upcoming matchups
  8. Attend focused practice sessions
  9. View tournament schedule in the system
  10. During the tournament, follow game progress and stats
    11. {% endif %}
Final Tip

Ultimate Coach is designed to be flexible - adapt these workflows to match your team's specific needs and coaching style.

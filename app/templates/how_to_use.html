Getting Started
{% if current_user.is_authenticated and current_user.is_admin %} {# ── ADMIN / COACH ONBOARDING ─── #}
Admin / Coach Setting up a new team

Follow these steps to go from a brand-new team to a fully set-up squad in minutes.

1
Create the team (superadmin only)
In the top-right user menu → Manage Teams → Add Team. Give it a name and division.
Manage Teams
2
Import your roster
Use Bulk Import to paste a spreadsheet of players in one go — far faster than adding one by one. The CSV format is: name, jersey, position, email, gender_match, line_preference.
Tick "Auto-create login accounts" to generate usernames and temporary passwords for all players with email addresses in the same step.
Bulk Import Players Add Single Player
3
Share login credentials with players
If you auto-created accounts in step 2, the results page shows a one-time credentials table — copy it and paste into a team chat or email. Players log in and can change their password from their profile. If you didn't create accounts yet, go to User Management to add or link accounts individually.
User Management
4
Choose which features to show
Toggle features on/off for your team so players only see what's relevant. Access this from the user menu → Feature Settings at any time.
Feature Settings
5
Add your first game or tournament
Once the roster is set, create an upcoming tournament or game. Players can RSVP, and you can start recording points and stats from game day.
Tournaments Games
As a superadmin you can switch between teams from the user menu → Switch Team. The blue team badge in the top-right shows which team you're currently managing — all data you add or view belongs to that team.
{% else %} {# ── PLAYER ONBOARDING ─── #}
Player Getting started as a player
1
Link your player profile
Your coach will have added you to the roster. Go to your profile to link your account to your player record so your stats and attendance appear on your dashboard.
My Profile
2
Check upcoming events
Your home page shows upcoming games, practices and tournaments. RSVP to let your coach know you're coming.
Home
3
Explore your stats and resources
Use the navigation above to explore statistics, training resources, and any other features your coach has enabled for the team.
The features available to you depend on what your coach has switched on. If you expect to see something that isn't there, ask your coach to enable it in Feature Settings.

Club & Organisation
Team Roster
Manage all player profiles, positions, and linked accounts.
  1. Go to Club → Team Roster
  2. Add players individually or use Import Players for bulk upload
  3. Click a player to view their profile and stats
Open Roster
Games
Record results, add points, and review post-game stats.
  1. Create a game with date, opponent, and location
  2. On game day use the Live Gameday tracker to record each point
  3. View full stats and analysis after the game
View Games
Tournaments
Organise weekend tournaments with RSVPs and squad selection.
  1. Create a tournament with dates and location
  2. Players RSVP — view attendance on the RSVP page
  3. Assign players to specific games within the tournament
View Tournaments
Team Calendar
See all games, practices and events in one calendar view.
  1. Browse upcoming events by month
  2. Click any event for details and RSVP options
Open Calendar

Game Day
Live Gameday Tracker
Record points in real time during a game, including lines, throws, and outcomes.
1
Open the game
Go to Club → Games, find the game you're about to play, and click "Start Gameday".
2
Select your line
Before each point, pick which 7 players are on the field. The system remembers frequent lines.
3
Record the point
Log throws, catches, turnovers and the point outcome. Stats update immediately.
4
Review post-game
After the game, the full stats dashboard is populated automatically. Share with the team from the Stats section.
Assign a dedicated stat-taker role so a coach or manager can focus on recording while others coach. You can set roles in User Management.

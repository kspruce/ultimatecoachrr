Option A — Webhook Setup

{# Step 1 #}
1 Open the channel you want notifications in

In Discord, navigate to the text channel where you'd like Ultimate Coach to post updates (e.g. #team-announcements or #schedule).

Tip: Create a dedicated channel like #uc-notifications to keep things tidy.
{# Step 2 #}
2 Open Channel Settings

Hover over the channel name in the left sidebar. Click the gear icon ⚙ that appears to the right of the name to open Edit Channel.

# team-announcements
📢 team-announcements   ⚙ ← click this
{# Step 3 #}
3 Go to Integrations → Webhooks

In the Channel Settings sidebar, click Integrations, then click the Webhooks button.

Channel Settings (left sidebar)
  • Overview
  • Permissions
  • → Integrations
  • Webhooks  click
  • Invites
{# Step 4 #}
4 Create a New Webhook

Click New Webhook. Discord will create a webhook called "Spidey Bot" — rename it to something like Ultimate Coach. You can also upload a logo if you like.

The webhook is already set to post to the channel you were in. You don't need to change the channel.
{# Step 5 #}
5 Copy the Webhook URL and paste it into Settings

Click Copy Webhook URL. The URL looks like this:

https://discord.com/api/webhooks/123456789/abcdefghijklmnopqrstuvwxyz

Go to Discord Settings, paste it into the Webhook URL field, enable the toggle, and click Save Settings.

That's it! Use the Send test message button to confirm it's working.

Option B — Discord Bot Setup

Before you start: You need to be a server administrator in Discord and have access to the Discord Developer Portal. You must also have a Discord account.
{# ── PART 1: Create the Application & Bot ── #}
Part 1Create the Application & Bot
1 Open the Discord Developer Portal

Go to discord.com/developers/applications and log in with your Discord account.

Click New Application (top-right). Give it a name — we suggest Ultimate Coach — and click Create.

2 Create the Bot user

In the left sidebar click Bot, then click Add Bot → Yes, do it!

You can give your bot a username and an icon here. These are what players will see in Discord.

3 Enable Privileged Intents

Still on the Bot page, scroll down to Privileged Gateway Intents and enable all three:

  • Presence Intent
  • Server Members Intent
  • Message Content Intent

Click Save Changes.

These are required so the bot can read messages and respond to !uc commands.
{# ── BOT TOKEN ── #}
4 Copy your Bot Token

On the Bot page, click Reset Token → Yes, do it! Discord will show your token once. Copy it immediately.

MTxxxxxxxxxxxxxxxxxxxxxxxx.Gxxxxx.xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
Keep this secret. Treat it like a password. Anyone with this token can control your bot. Never share it in a public Discord message or commit it to a code repository. Paste it into the Bot Token field in Discord Settings.
{# ── PART 2: Add Bot to Server ── #}
Part 2Add the Bot to Your Server
5 Generate an invite link

In the Developer Portal sidebar, go to OAuth2 → URL Generator.

Under Scopes, tick:

  • bot
  • applications.commands

Under Bot Permissions, tick:

  • Send Messages
  • Embed Links
  • Attach Files
  • Read Message History
  • Add Reactions
  • Manage Events
6 Invite the bot to your server

Scroll to the bottom of the URL Generator — you'll see a Generated URL. Copy it, open it in your browser, choose your server from the dropdown, and click Authorize.

You must be a server administrator to authorize the bot.
{# ── PART 3: Find IDs ── #}
Part 3Find Your Server & Channel IDs
7 Enable Developer Mode in Discord

You need Developer Mode on to copy IDs. It only takes a second:

  1. Open Discord (desktop or browser).
  2. Go to User Settings (gear icon bottom-left) → Advanced.
  3. Toggle Developer Mode on.
{# Server / Guild ID #}
8 Copy the Server (Guild) ID

Right-click on your server icon in the left panel (the circle with your server's name/logo).

A context menu appears — click Copy Server ID.

Context menu (right-click server icon):
  • Mark As Read
  • Mute Server
  • …
  • → Copy Server ID  click
1234567890123456789  (example — 18–20 digits)

Paste this into the Server (Guild) ID field in Discord Settings.

{# Channel IDs #}
9 Copy Channel IDs (Calendar & Notifications)

You need two channel IDs — one for calendar posts and one for general notifications. These can be the same channel if you prefer.

For each channel:

  1. Right-click the channel name in the sidebar.
  2. Click Copy Channel ID from the context menu.
  3. Paste it into the matching field in Discord Settings.
Context menu (right-click channel name):
  • Mark As Read
  • Mute Channel
  • …
  • → Copy Channel ID  click
Setting Which channel to use
Calendar Channel ID Channel where upcoming sessions/tournaments will be posted
Notification Channel ID Channel for general alerts (new sessions added, clips uploaded, etc.)
{# ── Bot Commands ── #}

Bot Commands Reference

Once connected, players can type these commands in any channel the bot can read.

Command What it does Who can use it
!uc help Shows all available commands Everyone
!uc upcoming Lists events in the next 7 days Everyone
!uc rsvp session 5 yes RSVP to a session (replace 5 with the session ID, and yes with no or maybe) Everyone
!uc rsvp tournament 3 no RSVP to a tournament Everyone
!uc link player@example.com Links your Discord account to your Ultimate Coach account so RSVPs are attributed correctly Everyone
{# ── Troubleshooting ── #}

Troubleshooting

  1. Make sure the Enable Discord notifications toggle is on and settings are saved.
  2. Double-check you pasted the full webhook URL — it should start with https://discord.com/api/webhooks/.
  3. Confirm the webhook still exists in Discord: Server Settings → Integrations → Webhooks.
  4. Check that the webhook's channel hasn't been deleted or restricted.

  1. Verify the bot was successfully added to your server (it should appear in the member list).
  2. Confirm the Message Content Intent is enabled in the Developer Portal → Bot → Privileged Gateway Intents.
  3. Check the bot has Send Messages and Read Message History permissions in the channel.
  4. Make sure your Bot Token is correct — if in doubt, reset it and paste the new one into Settings.

These options only appear when Developer Mode is on.

  1. Go to Discord → User Settings (gear icon) → Advanced.
  2. Toggle Developer Mode on.
  3. Now right-click the server icon or channel name again.

Immediately reset the token:

  1. Go to the Developer Portal.
  2. Select your application → Bot → Reset Token.
  3. Copy the new token and update it in Discord Settings.

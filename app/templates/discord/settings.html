Discord Bot Setup Guide
Step 1: Create a Discord Application
  1. Go to the Discord Developer Portal
  2. Click "New Application" and give it a name (e.g., "Ultimate Coach")
  3. Go to the "Bot" tab and click "Add Bot"
  4. Under the "Privileged Gateway Intents" section, enable:
    • Presence Intent
    • Server Members Intent
    • Message Content Intent
  5. Click "Reset Token" and copy the token (this is your Bot Token)
Step 2: Add Bot to Your Server
  1. Go to the "OAuth2" tab, then "URL Generator"
  2. Under "Scopes", select "bot" and "applications.commands"
  3. Under "Bot Permissions", select:
    • Send Messages
    • Embed Links
    • Attach Files
    • Read Message History
    • Add Reactions
    • Manage Events
  4. Copy the generated URL and open it in your browser
  5. Select your server and authorize the bot
Step 3: Get Server and Channel IDs
  1. In Discord, go to User Settings > Advanced and enable "Developer Mode"
  2. Right-click on your server icon and select "Copy ID" to get the Guild ID
  3. Right-click on the channels you want to use for calendar events and notifications and select "Copy ID" to get the Channel IDs
Discord Webhook Setup Guide
Creating a Discord Webhook
  1. Open Discord and go to the server where you want to add the webhook
  2. Go to Server Settings > Integrations > Webhooks
  3. Click "New Webhook"
  4. Give it a name (e.g., "Ultimate Coach Notifications")
  5. Select the channel where you want notifications to be sent
  6. Click "Copy Webhook URL" and paste it in the Webhook URL field above
  7. Click "Save Changes" in Discord
Webhooks are simpler to set up than bots but have fewer capabilities. They can only send messages to a specific channel and cannot respond to commands.

{# Step-by-step guide #}

How to get a Webhook URL (takes ~2 minutes)  ·  View full guide

  1. Open Discord and go to the channel where you want notifications to appear.
  2. Click the gear icon next to the channel name → Edit Channel.
  3. Select the Integrations tab → Webhooks → New Webhook.
  4. Give it a name (e.g. Ultimate Coach), then click Copy Webhook URL.
  5. Paste the URL into the field below and save.
Paste the webhook URL you copied from Discord. It stays hidden for security.

Discord Bot Setup Guide
Step 1 — Create a Discord Application
  1. Go to the Discord Developer Portal.
  2. Click New Application and name it (e.g. "Ultimate Coach").
  3. Go to the Bot tab → Add Bot.
  4. Enable Presence Intent, Server Members Intent, and Message Content Intent.
  5. Click Reset Token and copy it — this is your Bot Token.
Step 2 — Add the Bot to Your Server
  1. Go to OAuth2 → URL Generator.
  2. Select scopes: bot and applications.commands.
  3. Bot permissions: Send Messages, Embed Links, Read Message History, Manage Events.
  4. Open the generated URL, select your server and authorise.
Step 3 — Get Your Server and Channel IDs
  1. In Discord: User Settings → Advanced → Developer Mode (enable it).
  2. Right-click your server icon → Copy ID for the Guild ID.
  3. Right-click each channel → Copy ID for the channel IDs.

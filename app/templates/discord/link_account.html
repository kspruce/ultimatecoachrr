Link Your Discord Account
Linking your Discord account allows you to use Discord commands for RSVPs and receive notifications.
Option 1: Link via Discord Bot (Recommended)

Use the Discord bot command to link your account:

!uc link {{ current_user.email }}

Send this command in any channel where the Ultimate Coach bot is present.

Option 2: Enter Your Discord ID Manually

You can find your Discord ID by enabling Developer Mode in Discord settings, then right-clicking on your username and selecting "Copy ID".

Discord Bot Commands

Once your account is linked, you can use these commands in Discord:

Command Description
!uc upcoming Shows upcoming events for the next 7 days
!uc rsvp [event_type] [event_id] [response] RSVP to an event (session, tournament)
Responses: yes, no, maybe
Example: !uc rsvp session 5 yes

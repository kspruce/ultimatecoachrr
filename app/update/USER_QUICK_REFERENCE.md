# Clip Library - User Quick Reference

## For Players

### Viewing Clips and Annotations
1. Browse the **Clip Library** to find game footage and training videos
2. Click on any clip to watch and see annotations
3. Click on any timestamp to jump directly to that moment in the video
4. Use filters to find specific plays or moments

### Adding Your Own Annotations
1. While watching a clip, click **"Add New Annotation"**
2. Enter the timestamp (in seconds) where the play occurs
3. Give it a title (optional but helpful)
4. Select relevant tags (e.g., "Under Cut", "Break Throw")
5. Add any notes or observations
6. Tag yourself and teammates who were involved
7. Choose visibility:
   - **Team**: Everyone can see it
   - **Coaches**: Only coaches and admins see it
   - **Private**: Only you can see it

### Tips
- Use **Key Moment** checkbox for really important plays
- Tag multiple categories (e.g., both "Offense" and "Deep Cut")
- Be specific in your notes - help teammates learn!

## For Coaches

### Organizing Video Library
1. Upload clips via **"Add Clip"**
2. Tag clips by type:
   - **Full Game** - Complete game footage
   - **Training Session** - Practice footage
   - **Opponent Scouting** - Other teams' games
3. Associate clips with:
   - Specific games
   - Points within games
   - Players who appear

### Creating Teaching Annotations
1. Watch clips and pause at key moments
2. Add detailed annotations with:
   - **Event type** (goal, turnover, block, etc.)
   - **Score at the time**
   - **Offense/Defense type** being run
   - **Multiple tags** for searchability
   - **Coaching notes**
3. Mark **Key Moments** for plays you want to review later
4. Tag relevant players so they can find their own clips

### Film Session Workflow
1. **Before the session**:
   - Create a clip with the game footage
   - Add key annotations ahead of time
   - Mark the most important moments

2. **During the session**:
   - Click timestamps to jump to specific plays
   - Filter annotations by player or event type
   - Add new annotations based on team discussion

3. **After the session**:
   - Players can review annotations on their own
   - They can add their own observations
   - Track who has viewed and commented

### Tag Management
- Use hierarchical tags for organization
  - Example: Offense > Cutting > Under Cut
- Assign colors to categories for quick visual scanning
- Deactivate tags you're not using

### Best Practices
- **Be consistent** with tagging
- **Be specific** in notes (good: "Note the handler's timing on the reset" vs bad: "Good play")
- **Mark key moments** liberally - easy to filter later
- **Tag players** so they can find their own learning moments
- **Use visibility** wisely - use "Coaches" for corrective feedback, "Team" for teaching moments

## Tag Reference Guide

### Video Type Tags (Clip-Level)
When uploading a clip, categorize the footage:
- **Full Game** → Our Team vs Opponent / Opponent vs Opponent
- **Training Session** → Drills / Scrimmage / Theme-Based
- **Highlight / Tactic Showcase** → Set Plays / Strategy Review
- **Tournament Video**
- **Film Session**

### Annotation Tags (Tactical Analysis)

#### Offense
- **Handler Movement**: Reset, Upline, Give-and-Go
- **Cutting**: Under Cut, Deep Cut, Break Side Cut
- **Throw Types**: Backhand, Forehand, Hammer, High Release
- **Set Plays**: Horizontal Stack, Vertical Stack, Split Stack, Side Stack
- **Endzone Offense**

#### Defense
- **Person Defense**: Force Forehand/Backhand, Straight Up
- **Zone Defense**: 3-3-1 Cup, Wall, Junk
- **Help & Switches**: Poach, Flash Poach, Switch
- **Marks**: No Break Mark, Break Allowed
- **Turnovers**: Block, Handblock, Interception

#### Skills
- **Throwing**: Inside-Out, Outside-In, Break Throw, Huck
- **Catching**: Layout Catch, Toe-in Catch, High Point

#### Situations
- **Pulling**: Pull Hangtime, OOB Pull
- **Receiving Pull**: Centering Pass, Immediate Huck
- **Sideline**: Trap Sideline, Break Trap
- **Wind**: Upwind, Downwind, Crosswind
- **Weather**: Rain, Heat

#### Outcomes

- Turnover (Offensive)

#### Field Zones

- Mixed Line

#### Errors

## Common Scenarios

### "I want to review all my catches"
1. Go to Clip Library
2. Use player filter → select yourself
3. Use tag filter → select "Catching Skills"

### "I want to see how we defend zone"
1. Browse to game clips
2. Filter annotations by "Zone Defense"
3. Click timestamps to review each point

### "I want to study break throws"
1. Search annotations with "Break Throw" tag
2. Review notes from coaches
3. See which games/situations they occurred in

### "I want to create a highlight reel"
1. Use "Key Moments" filter
2. Note the clip IDs and timestamps
3. Create custom compilation outside the app (or request feature!)

### "I want to prepare for our next opponent"
1. Find clips tagged "Opponent Scouting"
2. Filter to the specific team
3. Review their offense/defense patterns

## FAQ

**Q: Can I delete someone else's annotation?**
A: Only coaches, admins, or the person who created it can delete annotations.

**Q: What's the difference between clip tags and annotation tags?**
A: Clip tags describe the video itself (e.g., "Full Game"). Annotation tags describe specific moments within the video (e.g., "Break Throw").

**Q: Can I annotate the same timestamp multiple times?**
A: Yes! Multiple people can add different perspectives on the same play.

**Q: What does "Key Moment" mean?**
A: It highlights particularly important plays that deserve extra attention during film review.

**Q: Should I tag every player in every annotation?**
A: Tag players who are directly involved in the play or who should learn from it.

**Q: How do I know what timestamp to use?**
A: Pause the video at the moment you want to annotate, note the time, then use that in seconds. For example, 1:45 = 105 seconds.
